## 🚀 Getting Started

1. **Open in Browser**:
   - Simply open `index.html` in any modern web browser
   - No server setup required

2. **Local Development**:
   - All files work locally without HTTP server
   - Can be used with any simple HTTP server:
     ```bash
     # Using Python 3
     python -m http.server 8000
     
     # Using Node.js (http-server)
     npx http-server
     ```

3. **Customization**:
   - Edit HTML files to change content
   - Modify `style.css` for color schemes
   - Update `script.js` for interactive features
   - Replace gradient backgrounds with real images
